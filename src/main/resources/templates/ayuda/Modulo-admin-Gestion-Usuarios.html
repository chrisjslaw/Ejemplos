Ayuda al usuario SGS




Módulo de administración:


Pantalla Administración de Usuarios


La pantalla muestra la tabla de usuarios con la información de cada uno.


1- Botón Crear usuario: desde aquí podremos acceder a crear un nuevo usuario en el sistema, ver prceso de creación de un nuevo usuario.

2- Botones exportación: permite copiar y exportar la información de la tabla usuarios a pdf y excel.

3- Campo de búsqueda para filtrar los datos por ejemplo: si escribimos chris solo aparecen los registros que contengan chris.

4- Botón editar: permite editar la información del usuario, dar de baja, cambiar el rol, ver proceso para editar un usuario.

5- Botón cambiar password: Permite cambiar el password/contraseña del usuario seleccionado, ver el proceso de cambio de contraseña.

6- Ordenar columnas: al darle clic a las flecha cambiará el orden de la columna si es texto se ordenará alfabeticamente y si es numérico cambiará de mayor a menor, lo mismo para las fechas.



7- Cantidad de registros: este mensaje muestra la cantidad de registros en la pagina de un total por ejemplo: del 1 al 10 de un total de 15 registros.

8- Botones para la navegación de registros: estos botones nos ayudan a navegar entre los registros con facilidad.


Formulario crear nuevo usuario


Proceso de creación de usuario.


Paso 1: primero rellenamos el campo Nombre Completo del usuario.


Paso 2: en el campo Nombre de Usuario introducimos el nombre a utilizar para iniciar sesión.


Paso 3: ingresamos la contraseña que utilizará el usuario para iniciar sesión, debajo de este campo se encuentra la opción mostrar, la cual permite ver ambas contraseñas en caso de que hayamos olvidado lo que escribimos.


Nota: La contraseña debe tener más de 8 caracteres y no debe contener espacios en blanco.


Paso 4: ingresamos de nuevo la contraseña para confirmar que es la misma que en el paso 3.


Paso 5: ingresamos la respuesta a la primer pregunta de seguridad, la cual nos ayudará a validar que el usuario es dueño de dicha cuenta, en el proceso de cambio de contraseña.


Paso 6: ingresamos la respuesta a la segunda pregunta de seguridad, la cual nos ayudará a validar que el usuario es dueño de dicha cuenta, en el proceso de cambio de contraseña.


Paso 7: cambiamos el campo al estado activo para que la cuenta pueda ser usada.


Paso 8: procedemos a asignarle un rol al usuario.


Roles:

Usuario: puede realizar requisiciones al administrador del sistema, puede ver el estado de sus requisiciones.

Gerente_Financiero: este usuario puede realizar requisiciones y aprobar requisiciones hechas por los demás usuarios.

Administrador: se encarga de gestionar el sistema y puede realizar reportes, realizar entradas y salidas(requisiciones), manejar proveedores, ingresar nuevos articulos, categorias y unidades de medida.

Super_Administrador:

tiene las mismas funciones que un administrador, pero puede gestionar los usuario y ver los registros de bitácora.


Paso 9: en caso de querer regresar al listado de usuarios sin terminar de guardar el usuario nuevo, podemos darle clic al boton volver.



Paso 10: Seleccionamos el departamento al que pertenece el usuario.


Paso 11: Introducimos el nombre del cargo del usuario.


Paso 12: Al rellenar todos los campos procedemos a darle click al boton crear usuario para finalizar la creación y guardarlo.


Formulario editar usuario


Proceso para editar la información de un usuario

Una vez que hayamos dado click al botón editar en un registro de la lista de usuarios, se cargara un formulario con la información de dicho usuario para que esta pueda ser editada.

Luego procederemos a realizar estos pasos para poder editar la información exitosamente.


Paso 1: Editamos los campos necesarios.


Nota: en caso de querer dar de baja a un usuario simplemente cambiamos el campo Estado al valor "Inactivo", de igual forma si lo queremos reactivar cambiaremos el estado a "Activo".


Paso 2: en caso de no querer guardar los cambios realizados y solo quiere regresar a la lista de usuarios, puede presionar el botón volver.


Paso 3: Damos click a editar usuario para finalizar el proceso.


Formulario editar contraseña


Proceso de cambio de contraseña

Una vez que hayamos dado click al botón cambiar en un registro de la lista de usuarios, se cargara un formulario que nos pedira la nueva contraseña para el usuario.

Luego procederemos a realizar estos pasos para poder cambiar la contraseña exitosamente.


Paso 1: Ingresamos la contraseña nueva.


Nota: Debajo de este campo tendremos la opción mostrar, la cual nos ayudara en caso de no recordar la contraseña que escribimos para poder confirmarla.


Paso 2: Confirmamos la contraseña nueva, es obligatorio que la contraseña que escribimos en el campo Nueva Contraseña sea igual al de Confirmar Nueva Contraseña.


Paso 3: En caso de querer cancelar el proceso podemos darle click al botón volver para regresar al listado de usuarios.


Paso 4: Una vez esten rellenados los campos necesarios procedemos a darle click al botón Guardar Contraseña para guardar la nueva contraseña del usuario.
