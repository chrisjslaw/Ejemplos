Módulo articulos: Pantalla gestión de unidades de medida


Pantalla Administración de Unidades de Medida

La pantalla muestra la tabla de unidades de medida con la información de cada una.

1- Botón Crear unidad de medida: desde aquí podremos acceder a crear un una nueva unidad de medida de articulo en el sistema, ver proceso de creación de unidad de medida.

2- Botones exportación: permite copiar y exportar la información de la tabla unidad de medida a pdf y excel.

3- Botón editar: permite editar la información del de la unidad de medida, ver proceso para editar la informacion de la unidad de medida.

4- Botón eliminar: permite eliminar la unidad de medida de esa fila, ver proceso para eliminar una unidad de medida.

5- Campo de búsqueda para filtrar los datos por ejemplo: si escribimos Resma solo aparecen los registros que contengan Resma.

6- Ordenar columna: al darle clic a las flecha cambiará el orden de la columna si es texto se ordenará alfabéticamente y si es numérico cambiará de mayor a menor, lo mismo para las fechas.

Formulario crear nueva unidad de medida

Proceso de creación de unidad de medida.

Paso 1: ingresamos el nombre de la unidad de medida.

Paso 2: ingresamos una breve descripción de la unidad de medida.

Paso 3: en caso de no querer continuar con el proceso de guardado, puede regresar a la lista de unidades de medida dandole clic al botón volver.

Paso 4: al finalizar de ingresar la información damos clic al botón crear Unidad para guardarlo.

Formulario editar unidad de medida

Proceso de editar información de una unidad de medida.

Paso 1: editamos los datos deseados.

Paso 2: en caso de no querer continuar con el proceso de guardado, puede regresar a la lista de Unidades de medida dandole clic al botón volver.

Paso 3: al finalizar de editar la información damos clic al botón editar unidad para guardar los cambios realizados.

Proceso eliminar una unidad

Clic al botón eliminar

Damos clic en el botón eliminar de la fila que deseamos borrar el registro.

Confirmar eliminación

Al darle clic al botón eliminar nos aparecerá un mensaje de confirmación, presionamos aceptar si lo deseamos eliminar, caso contrario le damos clic a cancelar.

Eliminación exitosa.

Después de aceptar eliminar el registro nos aparecerá el siguiente mensaje con el nombre de la Unidad de medida que fue eliminada con éxito siempre y cuando este no este ligado a otros registros.
