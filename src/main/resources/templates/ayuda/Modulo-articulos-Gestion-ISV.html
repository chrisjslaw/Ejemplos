Ayuda al usuario SGS




Módulo de artículos:

Pantalla editar ISV

En este formulario editaremos el campo ISV que representa el impuesto que manejaran las entradas de artículos.


Para ello realizaremos los siguientes pasos:

Paso 1: Llenamos el campo ISV con el valor de impuesto que se utilice actualmente en el pais por ejemplo: actualmente se utiliza el 15% entonces lo introducimos como 1.15, si el valor cambiara a 16% entonces lo introduciriamos como 1.16.

Paso 2: Si no quisieramos realizar ningún cambio o solamente quisieramos ver el valor que tiene el ISV podemos darle clic al botón volver, el cual nos regresa a la pantalla principal.

Paso 3: Si realizamos algún cambio y deseamos guardarlo, damos clic al botón Editar ISV.
